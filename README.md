# 아이온2 흑막 레기온 통합 정보실

## Ai Assistant : Claude Code

## 환경 설정

1. `.env.example` 파일을 복사하여 `.env` 파일 생성:

```bash
cp .env.example .env
```

2. 필요한 경우 `.env` 파일의 환경 변수 수정

## 환경 변수

- `VITE_AION2_API_URL`: 아이온2 검색 API 주소
- `VITE_AION2_PROFILE_IMAGE_URL`: 캐릭터 프로필 이미지 서버 주소

## 개발 서버 실행

```bash
npm run dev
```
